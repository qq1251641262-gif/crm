# CRM 客户管理系统

## 项目简介
这是一个基于 Web 的客户关系管理（CRM）系统，提供客户管理、合同管理、日程管理、数据分析等功能。

## 功能模块
- 📋 **客户管理** - 客户信息、联系人、跟进记录
- 📄 **合同管理** - 合同创建、跟踪、统计
- 📅 **日程管理** - 日程安排、提醒
- 📊 **数据分析** - 客户阶段分析、产品分析、目标分析
- 📧 **邮件管理** - 邮件发送、配置
- 👥 **组织架构** - 部门、职位、用户管理
- 🔐 **系统管理** - 角色权限、配置管理

## 🌐 在线访问

### 本地预览
- 地址：http://localhost:8313
- 说明：需要在本地运行 Python HTTP 服务器

### GitHub Pages 部署
1. 在 GitHub Desktop 中将此文件夹发布为仓库
2. 在 GitHub 仓库 Settings → Pages 中启用 GitHub Pages
3. 访问 `https://your-username.github.io/repo-name/`
